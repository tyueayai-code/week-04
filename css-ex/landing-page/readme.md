มีปุ่มให้กดสั่งซื้อ
กดเข้าไปมีภาพสินค้า ราคา ปุ่มกดถูกใจ แบ่งเป็นหมวดหมู่ ชุด เครื่องประดับ รองเท้า และ อื่น ๆ

.
### สรุปเนื้อหาสำคัญในไฟล์:
1. **Scope ชัดเจน:** ตัดส่วนของ JavaScript ออกทั้งหมด โดยเปลี่ยนปุ่มและตัวนับให้เป็นเลย์เอาต์โครงสร้าง HTML และ CSS พื้นฐานแทน
2. **โครงสร้างโค้ดแยกเป็น 2 ส่วนหลัก:**
   - โค้ด `index.html` (สำหรับจัดวางองค์ประกอบหน้าเว็บ ทั้งส่วนหัว, เรื่องราวความเป็นมา, หมวดหมู่สินค้า, รายการสินค้า, และข้อมูลติดต่อบริษัทท้ายเว็บ)
   - โค้ด `style.css` (การออกแบบสไตล์หรูหรา คุมโทนสีดาร์กไนท์และสีทอง ให้เข้ากับแบรนด์ชุดราตรี พร้อมใช้งานร่วมกับ CSS Grid/Flexbox)
3. **จัดระเบียบเนื้อหา:** เพิ่มสารบัญและข้อมูลติดต่อบริษัทไว้อย่างเป็นระบบ สามารถนำไปเปิดดูโครงสร้างหน้าตาเว็บไซต์บนเบราว์เซอร์ได้ทันที
